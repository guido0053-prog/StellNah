Dieser Ordner enthaelt alles, was du brauchst:

- backend/  -> "Motor" (sucht Stellplaetze, berechnet Fusswege)

REPLACE_WITH_YOUR_BACKEND_URL

Dann kannst du die Webseite (frontend) z.B. mit GitHub Pages hosten
und das Backend z.B. mit Render hosten.
